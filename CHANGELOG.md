# Changelog

## Последние изменения

### Интерактивное отображение данных через кнопки

**Что изменилось:**

#### ⚔️ Кнопка "Посмотреть экипировку"
- **Раньше:** Выводила текст "Используйте команду: /myequipment ID"
- **Теперь:** Сразу показывает всю экипировку персонажа с фотографиями
- После просмотра появляется кнопка "🔙 Назад к персонажу"

#### ⭐ Кнопка "Посмотреть желания"
- **Раньше:** Выводила текст "Используйте команду: /mywishes ID"
- **Теперь:** Сразу показывает все желания персонажа
- После просмотра появляется кнопка "🔙 Назад к персонажу"

#### 👥 Кнопка "Участники и экипировка"
- **Раньше:** Выводила текст "Используйте команду: /groupinfo ID"
- **Теперь:** Сразу показывает полную информацию о группе:
  - Список всех участников
  - Экипировка каждого участника
  - Желания каждого участника
- После просмотра появляется кнопка "🔙 К списку групп"

### Новые методы в CallbackQueryHandler

1. `showEquipmentForCharacter()` - показ экипировки персонажа
2. `showWishesForCharacter()` - показ желаний персонажа
3. `showFullGroupInfo()` - показ полной информации о группе

### Преимущества

✅ Меньше шагов для пользователя
✅ Вся информация доступна одним нажатием
✅ Удобная навигация назад через кнопки
✅ Единообразный интерфейс - все через кнопки

### Обратная совместимость

Текстовые команды `/myequipment`, `/mywishes`, `/groupinfo` продолжают работать как раньше.
